Boardgame Binder

[Matthew Lee/link to portfolio] created this MVP to showcase his full stack web dev skills. Boardgame Binder uses SQL, Express, Angular, and Node.js

Angular 5 template driven forms comprise most of the front-end. Guards and validation are in place to protect routes and data injection. Minimal CSS and Angular Material are used for aesthetics, with a mobile-first priority design.

The back end uses SQLite because of the the dataset source. Three tables are used in this relational DB in order to add boardgames to user's profile.
