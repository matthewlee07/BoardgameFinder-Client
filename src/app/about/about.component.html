About Boardgame Binder

Users search the library of nearly 10,000 games with filters such as title, number of players, and playing time.
After signing up for an account, users can record game experieces and share via social media. BoardGameGeek.

Matt Lee created this app to showcase his full stack abilites. Boardgame Binder uses Node, Express, PostgreSQL, Sequlize, AWS RDS, and Angular 5.
Read the Medium post of Matt's experience of hosting data on Amazon RDS.

Using App

On home page, click "Submit" to start a search for boardgames. Optional filters include keyword, number of players, playing time, and rating indirectly from BoardGameGeeks.
Click "more details" for game description and image. If already logged in, click "add game" to add game to personal binder; otherwise click "log in" to log in and then add game.
Click "Sign Up" or use Demo Account to use account able to add game. Enter information on sign up form. Username and email must be unique; password must be minimum 7 characters and include a letter and number.
When logged in, go to "My Binder" to manage personal games. Click "edit game" to save personal gameplay information. Click "remove game" to remove game from Binder.
